# Aditya Jha — Personal Portfolio

A responsive personal portfolio built with plain HTML, CSS, and JavaScript
(no frameworks). Built for CodSoft Web Development Internship — Level 1,
Task 1.

## Structure

```
.
├── index.html      # Page markup (header, about, skills, projects, resume, contact, footer)
├── style.css       # All styling — code-editor/terminal inspired theme
├── script.js       # Typed hero effect + scroll-based active tab highlighting
├── assets/
│   └── resume.pdf  # Replace with your own resume
└── README.md
```

## Sections

- **Header** — name and tagline, typed out terminal-style
- **About** — short bio and photo placeholder
- **Skills** — key technologies, styled as key/value pairs
- **Projects** — sample project cards (edit with your real projects)
- **Resume** — download link for a PDF resume
- **Contact** — email, phone, GitHub, LinkedIn
- **Footer** — copyright and back-to-top link

## Customize

1. Replace the placeholder name, bio, skills, and project details in `index.html`.
2. Swap `assets/resume.pdf` with your actual resume.
3. Update the social links in the Contact section.
4. Add a real photo by replacing `.avatar-placeholder` with an `<img>` tag.

## Run locally

Just open `index.html` in a browser — no build step required.

## Deploy for free (GitHub Pages)

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under "Source", pick the `main` branch and `/ (root)` folder.
4. Your site will be live at `https://<username>.github.io/<repo-name>/`.
